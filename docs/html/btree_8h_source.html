37 /* CONVENTIONS: All data structures for red-black trees have the prefix */
38 /* "rb_" to prevent name conflicts. */
39 /* */
40 /* Function names: Each word in a function name begins with */
41 /* a capital letter. An example funcntion name is */
42 /* CreateRedTree(a,b,c). Furthermore, each function name */
43 /* should begin with a capital letter to easily distinguish */
44 /* them from variables. */
45 /* */
46 /* Variable names: Each word in a variable name begins with */
47 /* a capital letter EXCEPT the first letter of the variable */
48 /* name. For example, int newLongInt. Global variables have */
49 /* names beginning with "g". An example of a global */
50 /* variable name is gNewtonsConstant. */
51
